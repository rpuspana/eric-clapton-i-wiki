Eric Clapton interactive wikipedia

To play this game, bring up your browser's console (the instructions are below).
Then click "Start game" to start playing.


Chrome
Windows: Ctrl + Shift + J
Mac OS: Cmd + Opt + J

Firefox
Windows: Ctrl + Shift + K
Mac OS: Cmd + Opt + K

Internet Explorer
F12, then click on the “Console” tab

Safari
Cmd + Opt + C
